# [T-001] 脚手架 — 构思

## 摘要

- **本 T**：项目脚手架初始化，配置小程序基础结构与全局样式。
- **做**：新写 app 配置与全局样式；验证可启动
- **不做**：业务逻辑、接口对接
- **上游**：REQ-001
- **AC**：AC-001

## 主流程

> 入口：项目初始化脚手架

1. 开发启动 → 配置 TabBar → `src/config/app.ts` 注册 pages
2. TabBar 完成 → 写 `src/styles/variables.wxss` 全局样式变量
3. 样式就绪 → `App.start()` 验证编译无错

## 边界

- **JSON 解析失败**：第 1 步 → 提示格式错误

## 实现说明

### `src/config/app.ts` 【新写】

- **目的**：TabBar 与页面路由
- **做什么**：注册 pages 与 tabBar.items
- **怎么做**：按页面清单写入 app.json 等价配置

### `src/styles/variables.wxss` 【新写】

- **目的**：全局样式变量
- **做什么**：定义 CSS 变量
- **怎么做**：从设计稿色值/字号写入

### `App.start()` 【新写】

- **目的**：验证脚手架
- **做什么**：启动 dev server
- **怎么做**：运行编译 → exit 0
