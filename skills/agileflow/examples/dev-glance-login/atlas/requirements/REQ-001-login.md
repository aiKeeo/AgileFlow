# [REQ-001] 账号密码登录

- 版本：0.2
- 状态：已确认
- 平台：微信小程序

## 用户故事

作为用户，我想用账号密码登录，以便进入首页并保持登录态。

## 范围提示（供 sol 提炼边界）

- 范围外：注册、第三方登录、refresh token
