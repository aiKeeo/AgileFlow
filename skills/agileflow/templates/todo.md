# todo.md 模板

```markdown
# 项目待办事项

## 项目信息
- 项目名称：
- 创建时间：
- 预计完成时间：

## 流程进度
- [ ] 需求澄清
- [ ] 数据建模
- [ ] 技术方案
- [ ] 开发实现
- [ ] 测试验收

## 开发任务
- [ ] 任务1：创建数据库表（30min）

## 进行中
- 无

## 已完成任务
- 无

## 变更历史
| 时间 | 操作 | 说明 |
|------|------|------|
```

## 状态标记

| 标记 | 含义 |
|------|------|
| `- [ ]` | 未开始 |
| `- [ ] 🔄` 或「进行中」区 | 进行中 |
| `- [x]` / `✅` | 已完成 |

任务完成后从「开发任务」移到「已完成任务」，追加变更历史。
