# 需求文档输出模板

## REQ 文档（`specs/requirements/REQ-XXX-名称.md`）

```markdown
# [REQ-XXX] 需求名称

- 版本：1.0
- 创建时间：{{当前时间}}
- 状态：草稿 | 已确认 | 已实现 | 已废弃
- 关联：无（或填写其他需求编号）

## 业务价值与用户角色
- 目标用户：……
- 业务价值：……

## 用户故事
作为 [角色]，我想 [功能]，以便 [价值]。

## BDD 验收场景

### 场景1：正常流程
**Given** …
**When** …
**Then** …

### 场景2：异常流程
**Given** …
**When** …
**Then** …

### 场景3：边界条件
**Given** …
**When** …
**Then** …

## 非功能需求（可选）
- 性能：……
- 安全：……

## 版本历史
| 版本 | 日期 | 变更人 | 变更说明 |
|------|------|--------|----------|
| 1.0  | {{当前时间}} | 系统 | 初始创建 |
```

## 索引（`specs/requirements/README.md`）

```markdown
# 需求索引

| 编号 | 名称 | 版本 | 状态 | 文件 |
|------|------|------|------|------|
| REQ-001 | 用户登录 | 1.0 | 已确认 | REQ-001-用户登录.md |
```
