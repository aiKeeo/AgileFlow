# [REQ-001] 账号密码登录

- 版本：0.2
- 状态：已确认
- 平台：微信小程序

## 用户故事

作为用户，我想用账号密码登录，以便进入首页并保持登录态。

## 范围提示（供 sol 提炼边界）

- 范围内：账号密码登录并返回 JWT token 给调用方使用，以及登录页提交与跳转
- 范围外：注册流程、第三方 OAuth、找回密码与 refresh token 轮换
