## 状态标记

| 标记 | 含义 |
|------|------|
| 三步子项 `- [ ]` | 该步未开始/进行中 |
| 三步子项 `- [x]` | 该步已完成 |
| 父任务 `- [x]` | ①②③ 三步全 ✅ |
| 「进行中」区 | 当前正在做的 **T-xxx**（含当前步 ①/②/③） |

## 规则

- 开发任务只维护此处；**一任务 = 一个 `### T-xxx` 头 + 三步子项**
- 阶段 3：拆解任务 + 功能依赖；**每个 T-xxx 须预填三步子项与 dev 路径**
- 阶段 4：**严格按子项顺序** — 勾 ① 并落盘 dev → 才允许勾 ② 写码 → 才允许勾 ③ 验收 → 父任务 ✅（细则 [04-development.md](../phases/04-development.md)）
- 阶段 4 开始 **T-xxx** 时：**必须 `TodoWrite` 同步该任务三步**；会话内与 `atlas/todo.md` 同步勾选
- **禁止**：跳过 ① 直接写码；三步未全 ✅ 就把父任务标完成
